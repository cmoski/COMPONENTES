Easy financial chart Windows Demo Hot Keys:

Chart movement:
Left Arrow Move cursor left by one day
Right Arrow Move cursor right by one day
Ctrl + Left Arrow Move cursor left by ten days
Ctrl + Right Arrow Move cursor right by ten days
Alt + Left Arrow Move chart left by one day
Alt + Right Arrow Move chart right by one day
Ctrl + Alt + Left Arrow Move chart left by ten days
Ctrl + Alt + Right Arrow Move chart right by ten days
Home Move chart to the first day
End Move chart to the latest day
Up Arrow Expand the candle width
Down Arrow Shrink the candle width
F8 Switch data cycle
* / switch indicators
Ctrl+Down Arrow move selected area down
Ctrl+Up Arrow move selected area up

Shortcut Keys:
Ctrl + O Open stock data from file
Ctrl + J Open calculator
Ctrl + N Change axis type to normal
Ctrl + L Change axis type to logarithm
Alt + F4 Close the demo program

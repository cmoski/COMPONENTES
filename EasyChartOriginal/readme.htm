Microsoft.Web.UI.WebControls.dll : Free IE WebControl from microsoft, Used in WebDemos, for more information about these controls please go to http://msdn.microsoft.com/downloads/samples/internet/asp_dot_net_servercontrols/webcontrols/default.asp

1. How to use the web demos:

(1) Map the Webdemos folder in your IIS server.
(2) Xcopy "webctrl_client" folder to your web server root.
(3) Visit the demos pages at your web browser.

2. How to use the web DB demo:
Web DB demo guide

3. How to use the windows demo:
Windows demo guide

4. How to write you own formulas:
Write you own formulas

5. How to open web demos at visual studio.

Simply open WebSolution.sln by visual studio.

6. How to open windows demo at visual studio.

Simply open WinSolution.sln by visual studio.

7. How to open formula editor source code at visual studio.

Simply open CompilerSolution.sln by visual studio.

8. How to add reference to your visual studio project.

(1) Right click your project reference folder.
(2) Add bin\easychart.finance.dll.
(3) Add line "using Easychart.Finance;" at the top of you program.
